# YouTube 音樂播放器使用指南

## 快速開始

### 1. 獲取 YouTube API Key

在使用本應用程式之前，您需要獲取 YouTube Data API v3 的 API Key：

#### 步驟：

1. **前往 Google Cloud Console**
   - 訪問：https://console.cloud.google.com/

2. **創建或選擇專案**
   - 點擊頂部的專案選擇器
   - 選擇現有專案或點擊「新增專案」

3. **啟用 YouTube Data API v3**
   - 在左側選單中選擇「API 和服務」→「程式庫」
   - 搜尋「YouTube Data API v3」
   - 點擊該 API 並點擊「啟用」

4. **創建憑證**
   - 選擇「API 和服務」→「憑證」
   - 點擊「建立憑證」→「API 金鑰」
   - 複製生成的 API 金鑰

5. **設定 API Key**
   - 打開 `widget.cpp` 檔案
   - 找到第 20 行左右的 `apiKey` 變數
   - 將 `YOUR_API_KEY_HERE` 替換為您的 API Key
   
   ```cpp
   apiKey("YOUR_ACTUAL_API_KEY_HERE")
   ```

### 2. 編譯應用程式

#### Windows (使用 Qt Creator):
1. 打開 Qt Creator
2. 開啟 `last-report.pro` 檔案
3. 選擇 Qt 6.x 的 Kit
4. 點擊「執行」或按 Ctrl+R

#### Linux/macOS (命令列):
```bash
# 確保已安裝 Qt 6 和 Qt WebEngine
qmake6 last-report.pro
make
./last-report
```

## 功能詳解

### 搜尋影片

1. **輸入搜尋關鍵字**
   - 在頂部搜尋欄輸入歌曲名稱、藝術家或任何關鍵字
   - 例如：「周杰倫 晴天」、「Taylor Swift」、「lofi hip hop」

2. **執行搜尋**
   - 點擊「🔍 搜尋」按鈕
   - 或直接按 Enter 鍵

3. **查看結果**
   - 搜尋結果會顯示在底部列表中
   - 每個結果顯示影片標題和頻道名稱

4. **播放搜尋結果**
   - 雙擊任一搜尋結果即可立即播放
   - 或選擇結果後點擊「➕ 加入」添加到播放清單

### 播放控制

#### 基本播放
- **▶ 播放**: 播放當前選中的影片或播放清單第一首
- **⏸ 暫停**: 暫停當前播放（目前簡化實作）
- **⏮ 上一首**: 播放播放清單中的上一首
- **⏭ 下一首**: 播放播放清單中的下一首

#### 播放模式
- **🔀 隨機播放**:
  - 點擊啟用/停用隨機播放
  - 啟用時按鈕變為綠色
  - 會記錄已播放的影片，避免重複

- **🔁 循環播放**:
  - 點擊啟用/停用循環播放
  - 啟用時播放清單結束後會重新開始
  - 配合隨機模式使用效果更佳

### 播放清單管理

#### 創建播放清單
1. 點擊左側的「➕ 新增」按鈕
2. 輸入播放清單名稱
3. 新播放清單會自動選中

#### 添加影片到播放清單
1. 搜尋並找到想要的影片
2. 在搜尋結果中選中該影片
3. 點擊「➕ 加入」按鈕
4. 影片會添加到當前選中的播放清單

#### 移除影片
1. 在播放清單中選中要移除的影片
2. 點擊「➖ 移除」按鈕
3. 確認移除

#### 切換播放清單
- 使用播放清單下拉選單選擇不同的播放清單
- 應用程式會記住上次使用的播放清單

#### 刪除播放清單
1. 選擇要刪除的播放清單
2. 點擊「🗑️ 刪除」按鈕
3. 確認刪除
4. 注意：至少需要保留一個播放清單

### 我的最愛功能

#### 加入最愛
1. 播放任一影片
2. 點擊「❤️ 加入最愛」按鈕
3. 影片會自動添加到「我的最愛」播放清單

#### 移除最愛
1. 播放已加入最愛的影片
2. 點擊「💔 移除最愛」按鈕
3. 影片會從「我的最愛」播放清單中移除

### 背景播放

- 將視窗最小化後，影片仍會繼續播放
- 可以在做其他事情時持續聆聽音樂
- 使用系統音量控制調整音量

## 鍵盤快捷鍵

目前版本沒有特別定義鍵盤快捷鍵，但以下操作可以使用鍵盤：

- **搜尋欄**: Enter 鍵執行搜尋
- **播放清單**: 上下箭頭鍵選擇，Enter 鍵播放
- **搜尋結果**: 上下箭頭鍵選擇，Enter 鍵播放

## 常見問題

### Q: 搜尋沒有結果？
A: 請檢查：
- 網路連線是否正常
- API Key 是否正確設定
- YouTube API 配額是否已用完（每天 10,000 單位）

### Q: 影片無法播放？
A: 可能原因：
- 網路速度不足
- 該影片可能有地區限制
- 影片可能已被刪除或設為私人

### Q: API 配額用完了怎麼辦？
A: 
- YouTube API 配額每天午夜（太平洋時間）重置
- 可以創建多個 Google Cloud 專案使用不同的 API Key
- 考慮申請增加配額（需要付費）

### Q: 可以下載影片嗎？
A: 
- 本應用程式不支援也不建議下載影片
- 請尊重內容創作者的版權
- 僅供線上串流播放使用

### Q: 播放清單資料儲存在哪裡？
A: 
- Windows: `C:\Users\<username>\AppData\Local\<AppName>\youtube_playlists.json`
- Linux: `~/.local/share/<AppName>/youtube_playlists.json`
- macOS: `~/Library/Application Support/<AppName>/youtube_playlists.json`

## 使用技巧

### 1. 建立主題播放清單
- 為不同心情或場合創建專門的播放清單
- 例如：「工作專注音樂」、「運動音樂」、「放鬆音樂」

### 2. 善用隨機播放
- 在大型播放清單中啟用隨機播放
- 發現被遺忘的好歌

### 3. 快速播放
- 雙擊搜尋結果可以立即試聽
- 喜歡的話再加入播放清單

### 4. 最愛清單
- 將真正喜歡的歌曲加入最愛
- 作為快速訪問的精選集

## API 使用限制

### 配額說明
- 每個 API Key 每天有 10,000 單位的配額
- search 操作：100 單位/次
- 理論上每天可以搜尋 100 次

### 節省配額的方法
- 精準使用搜尋關鍵字
- 避免重複搜尋相同內容
- 多使用播放清單功能
