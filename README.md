# 音樂播放器 (Music Player)

這是一個使用 Qt6 C++ 開發的功能豐富的音樂播放器。

## 功能特色

### 1. 播放清單管理
- 使用播放清單來管理多首歌曲
- 提供增加、刪除、調整順序等功能
- 支援多個播放清單的建立與切換
- 用戶可以根據需求，自由調整歌曲的播放順序

### 2. 上/下首歌曲控制
- 利用 QMediaPlayer 實現「上一首」與「下一首」的功能
- 支援循環播放（自動回到開頭/結尾）
- 直觀的操作按鈕設計

### 3. 播放進度顯示
- 透過 QSlider 控制播放進度
- 使用 positionChanged 來即時更新播放位置
- 以格式化的時間（mm:ss）顯示播放進度和總時長
- 支援拖曳滑桿快速跳轉播放位置

### 4. 歌曲封面展示
- 自動搜尋並顯示歌曲所在目錄的封面圖片
- 支援 cover.jpg、album.png 等常見封面文件名
- 封面圖片會隨著歌曲切換自動更新

### 5. 播放器控制介面
- 播放/暫停按鈕
- 停止播放按鈕
- 上一首/下一首按鈕
- 音量控制滑桿
- 簡潔直觀的介面設計

### 6. 多格式支援
- MP3 (MPEG Audio)
- WAV (Waveform Audio)
- FLAC (Free Lossless Audio Codec)
- AAC (Advanced Audio Coding)
- OGG (Ogg Vorbis)
- M4A (MPEG-4 Audio)
- WMA (Windows Media Audio)

## 操作說明

### 基本操作
- **▶ 播放/⏸ 暫停**: 開始或暫停播放當前歌曲
- **⏹ 停止**: 停止播放並重置進度
- **⏮ 上一首**: 切換到上一首歌曲
- **⏭ 下一首**: 切換到下一首歌曲
- **進度滑桿**: 拖曳可跳轉到指定位置
- **音量滑桿**: 調整播放音量 (0-100%)

### 播放清單管理
- **新增**: 創建新的播放清單
- **刪除**: 刪除當前播放清單
- **添加歌曲**: 從本地選擇音樂文件添加到清單
- **移除歌曲**: 從清單中移除選中的歌曲
- **上移/下移**: 調整歌曲在清單中的順序
- **雙擊歌曲**: 立即播放該歌曲

## 編譯和運行

### 系統需求
- Qt 6.x 或更高版本
- Qt Multimedia 模組
- C++17 編譯器

### 安裝依賴 (Ubuntu/Debian)
```bash
sudo apt-get install qt6-base-dev qt6-multimedia-dev
```

### 編譯
```bash
qmake6 last-report.pro
make
```

### 運行
```bash
./last-report
```

## 技術實現

### 核心類別
- **QMediaPlayer**: 音頻播放核心
- **QAudioOutput**: 音頻輸出控制
- **QSlider**: 進度和音量控制
- **QListWidget**: 播放清單顯示
- **QComboBox**: 播放清單選擇

### 信號與槽
- `positionChanged`: 更新播放進度
- `durationChanged`: 更新歌曲總時長
- `mediaStatusChanged`: 監控播放狀態
- `playbackStateChanged`: 更新播放按鈕狀態

## 界面預覽

播放器界面包含：
- 歌曲封面顯示區域 (200x200 像素)
