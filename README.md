# 麻將配對遊戲 (Mahjong Matching Game)

這是一個使用 Qt C++ 開發的簡單麻將配對遊戲。

## 遊戲說明

### 遊戲目標
配對所有相同的麻將牌，直到遊戲板清空為止。

### 遊戲規則
1. 遊戲板上有 32 張牌（16 對）
2. 點擊兩張相同的牌進行配對
3. 配對成功的牌會變成綠色並被移除
4. 配對失敗時，牌會恢復原狀，需要重新選擇
5. 完成所有配對即可獲勝

### 麻將牌類型
- **萬子**: 1萬 到 9萬
- **筒子**: 1筒 到 9筒  
- **索子**: 1索 到 9索
- **字牌**: 東、南、西、北、中、發、白

### 操作說明
- **點擊牌**: 選擇要配對的牌
- **重新開始**: 點擊「重新開始」按鈕開始新遊戲

## 編譯和運行

### 需求
- Qt 6.x 或更高版本
- C++17 編譯器

### 編譯
```bash
qmake last-report.pro
make
```

### 運行
```bash
./last-report
```

## 遊戲特色
- 簡單直觀的遊戲界面
- 傳統麻將牌設計
- 即時配對反饋
- 分數追蹤系統
